# RF Based War Spying Robot with Wireless Night Vision Camera

## Objective
This project is designed for surveillance in dangerous environments using RF communication and live video streaming.

## Features
- Long-range RF control
- Wireless night vision camera
- GPS and GSM alerts
- Obstacle detection
- Gas and metal detection

## Components Used
- Arduino Mega
- Arduino Uno
- HC-12 RF Module
- GSM Module
- GPS Module
- Ultrasonic Sensor
- IP Camera

## Applications
- Military surveillance
- Border security
- Disaster management
